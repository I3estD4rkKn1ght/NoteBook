# 数据类型
## 初识数据类型
* 整型：数字
* 字符串：文本
* 布尔类型：真假
## 整型
存储数字
```go
package main

import "fmt"

func main() {
    fmt.Println(666)
    fmt.Println(6 + 9)
    fmt.Println(6 - 9)
    fmt.Println(6 * 9)
    fmt.Println(9 / 6)
    fmt.Println(9 % 6)
    // 加减乘除求余  
}
```
## 字符串
文本 双引号中间
```go
package main

import "fmt"

func main() {
    fmt.Println("hello,world")
    fmt.Println("hello" + "world")
    // 字符串和数字不能相加 类型不能混搭用
}
```
## 布尔类型
真假 true false
```go
package main

import "fmt"

func main() {
    fmt.Println(1 > 2) // 布尔值 false
    fmt.Println(1 < 2) // true
}
```
条件语句 经常用到 用于判断
